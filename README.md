# Pong Game
A browser-based implementation of the classic Pong arcade game built using HTML, CSS, and JavaScript. The game runs directly in the browser and includes simple controls, sound effects, and a start/restart system.

## Overview

This project recreates the classic Pong gameplay where a player controls a paddle to bounce a ball and score points. It demonstrates basic game mechanics, interactive controls, and dynamic updates using JavaScript.

## Features

* Interactive Pong gameplay
* Player paddle control
* Ball movement and collision detection
* Score tracking
* Sound effects for paddle hit and scoring
* Start and restart game functionality

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run the Project

1. Clone the repository
2. Open the project folder
3. Open `index.html` in a web browser
4. Click **Start** to begin playing
